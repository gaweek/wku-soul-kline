## 📚 使用说明

### 智能八字排盘

现在无需去外部网站查询！直接在应用中：

1. **填写出生信息**：
   - 选择出生日期（阳历）
   - 选择出生时间（24小时制）
   - 选择出生城市（用于真太阳时修正）

2. **自动计算**：
   - 系统自动计算四柱八字
   - 自动计算大运信息
   - 显示农历、节气等详细信息

3. **配置API并生成**：
   - 配置你自己的 API Key
   - 点击"生成人生K线"

## ✨ 功能特点

1. **🎯 智能八字排盘**: 只需填写出生年月日时和地点，系统自动计算准确八字，无需外部查询
   - 基于 lunar-javascript 库，精确计算农历、节气、干支
   - 自动真太阳时修正（根据出生地经度调整）
   - 自动计算大运排盘（顺逆行规则）
   - 支持手动输入模式（高级用户）

2. **可视化运势**: 结合金融技术分析，用股票 K 线图展示 1-100 岁的人生运势起伏，直观呈现人生"牛市"与"熊市"。

3. **AI 深度批断**: 利用大模型强大的推理能力，生成性格、事业、财富、婚姻、健康、六亲及发展风水等多维度报告。

4. **发展风水**: 提供详尽的方位建议、地理环境选择（如沿海、山区）及开运布局。

5. **Web3 特供**: 专为加密货币交易者设计的"币圈交易运势"板块，包含暴富流年预测与适合的交易风格建议。

6. **完全自包含**: 所有功能内置，无需跳转外部网站，保护用户隐私

## 🚀 快速开始

### 1. 配置环境变量

```bash
# 复制配置模板
cp .env.example .env
cp server/.env.example server/.env

# 编辑 .env 文件，填入你的 API Key
```

**必需配置项：**
- `API_KEY`: 你的 OpenAI 或兼容 API 的密钥
- `JWT_SECRET`: 用于用户认证的安全密钥（请使用随机字符串）

### 2. 安装依赖

```bash
npm install
# 或者
pnpm install
# 或者
yarn install
```

### 3. 启动开发服务器

```bash
# 启动前端
npm run dev

# 启动后端 (另一个终端)
cd server && node index.js
```

打开浏览器访问 `http://localhost:5173` 即可使用。

### 4. 构建生产版本

```bash
npm run build
```

## 🛠️ 技术栈

- **前端框架**: React 19 + Vite
- **UI 样式**: TailwindCSS
- **图表库**: Recharts
- **八字计算**: lunar-javascript (精确的农历和八字计算库)
- **AI 模型**: 支持 OpenAI 兼容的 API (GPT, Claude, Gemini 等)
- **后端**: Node.js + Express + SQLite

## 📋 API 兼容性

本项目支持任何 OpenAI 兼容的 API 接口，包括：
- OpenAI GPT-4 / GPT-3.5
- Anthropic Claude
- Google Gemini (通过兼容接口)
- 其他第三方 API 服务

只需在 `.env` 中配置 `API_BASE_URL` 和 `API_KEY` 即可。
