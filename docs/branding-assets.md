# 品牌视觉设计方案 (Branding Assets)

## 1. 设计理念
**“人生K线” (Life K-Line)** 是一个结合**传统命理文化**与**现代金融可视化**的产品。品牌视觉需要同时传达：
- **命运的流动性**（传统、玄学、曲线、S形）
- **数据的精准性**（现代、科技、K线、直角）
- **向上的希望感**（增长、阳线、未来）

## 2. Logo 图形设计
### 核心意象： “太极趋势” (The Taiji Trend)
将中国传统的“太极 S 曲线”与金融市场的“K 线柱”巧妙融合。

- **形态**：
  - 三根 K 线柱体（Candlesticks）按阶梯状排列，象征人生的起伏与成长。
  - 第一根（过去）：短小的阴线（实心/灰色），代表积累或低谷。
  - 第二根（现在）：中等长度的阳线（空心/过渡色），代表转折。
  - 第三根（未来）：长势强劲的阳线（主色），直冲云霄，代表人生巅峰。
  - **点睛之笔**：一条柔和的 S 形曲线（命运线）贯穿这三根 K 线，既像太极的分割线，又像均线系统，将离散的 K 线串联成连续的命运。

- **配色**：
  - **主色 (Primary)**：`#4F46E5` (Indigo-600) —— 象征智慧、玄学、深邃（现有品牌色）。
  - **辅色 (Accent)**：`#10B981` (Emerald-500) —— 象征生长、财富、阳线。
  - **背景**：透明或纯白/纯黑（适应 x.com 风格的深色模式）。

## 3. 字体设计 (Typography)
### “人生K线”
为了体现“中西合璧”，字体设计采用**“人文衬线 + 几何科技”**的混合风格。

- **“人生”**：
  - 基于宋体/明体结构，但削弱三角形装饰，笔画末端更加利落。
  - 寓意：传统底蕴，命理根基。
- **“K线”**：
  - 基于几何无衬线体（Geometric Sans），线条粗细均等，棱角分明。
  - “K” 的两笔做断开处理，像两根相交的趋势线。
  - 寓意：现代工具，理性分析。

- **Header**：使用 `logo-full.svg`，高度建议 `32px` - `40px`。
- **Favicon**：使用 `logo-icon.svg`。
- **社交媒体头像**：使用 `logo-icon.svg` 配合纯白或 Indigo 背景。
